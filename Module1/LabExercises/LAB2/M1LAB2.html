Commonly Used HTML Elements

  • <p> (Paragraph)

    The <p> element defines a paragraph of text.

    <p>This is a paragraph.</p>

    Example:

    This is a paragraph.

  • <a> (Anchor)

    The <a> element defines a hyperlink.

    <a href="https://www.example.com">Visit Example</a>

    Example:

    Visit Example
  • <img> (Image)

    The <img> element embeds an image in the document.

    <img src="image.jpg" alt="Description">

    Example:

  • <ul> and <li> (Unordered List and List Item)

    The <ul> element defines an unordered list, and the <li> element defines a list item.

    <ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

    Example:

    • Item 1
    • Item 2
    • Item 3
  • <ol> and <li> (Ordered List and List Item)

    The <ol> element defines an ordered list, and the <li> element defines a list item.

    <ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

    Example:

    1. First item
    2. Second item
    3. Third item
  • <h1> to <h6> (Headings)

    The <h1> to <h6> elements define HTML headings, with <h1> being the highest level and <h6> the lowest.

    <h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

    Example:

    Heading 1

    Heading 2

    Heading 3

    Heading 4

    Heading 5
    Heading 6
  • <div> (Division)

    The <div> element is a container for other HTML elements, used to group them together.

    <div>
    <p>This is a paragraph inside a div.</p>
</div>

    Example:

    This is a paragraph inside a div.

  • <span> (Span)

    The <span> element is an inline container for text and other inline elements.

    <span>This is a span.</span>

    Example:

    This is a span.
  • <table>, <tr>, <td>, <th> (Table, Table Row, Table Data, Table Header)

    The <table> element defines a table, <tr> defines a row, <td> defines a cell, and <th> defines a header cell.

    <table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

    Example:

    Header 1 Header 2
    Data 1 Data 2
  • <form>, <input>, <label> (Form, Input, Label)

    The <form> element defines a form, <input> defines an input field, and <label> defines a label for an input field.

    <form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Submit">
</form>

    Example:

  • <button> (Button)

    The <button> element defines a clickable button.

    <button>Click Me</button>

    Example:

  • <blockquote> (Blockquote)

    The <blockquote> element defines a section that is quoted from another source.

    <blockquote>This is a blockquote.</blockquote>

    Example:

    This is a blockquote.
